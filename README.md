# TalentMatch AI: Automated CV Screening & Matchmaking System

TalentMatch AI adalah sistem cerdas yang membantu rekruter menyaring dan mencocokkan CV kandidat dengan kriteria lowongan pekerjaan secara otomatis menggunakan teknologi *Natural Language Processing* (NLP) dan *Optical Character Recognition* (OCR).

## Fitur Utama
* **CV Extraction:** Mengekstrak teks dari dokumen CV (format gambar) menggunakan PaddleOCR.
* **AI Matchmaking:** Menghitung skor kecocokan antara pengalaman kandidat dan kriteria lowongan menggunakan model IndoBERT.
* **User-Friendly Dashboard:** Antarmuka web intuitif berbasis Streamlit untuk memproses data secara *real-time*.
